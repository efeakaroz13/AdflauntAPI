  • 'email'
  • 'password'

"password":password,
"ip":ip_,

/api/listing?id=PID GET
Returns the same thing as the input other than email and password

/api/listing DELETE
Arguments as form:
email,password,id
Returns just success value. Deletes the listing, anyone other than user can't delete the listing.

/search?q=search GET
returns searched listings
if you don't specify anything, it will return all
